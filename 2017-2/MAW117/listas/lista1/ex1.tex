Considere as seguintes funções, todas de $\R$ em $\R$:
\[
	f(x)=x^2+1;g(x)=\sqrt{|x|};h(x)=\left\{\begin{array}{l} x+1,\mbox{ se $x\geq 1$};\\ x-1,\mbox{ se $x<1$}.\end{array}\right.
\]
Calcule:
\begin{enumerate}
	\item $f(0)$;
	\item $g(4)$;
	\item $h(0)$;
	\item $h(3/2)$;
	\item $(g\circ f)(x)$;
	\item $(f\circ g)(x)$;
	\item $(h\circ g\circ f)(x)$;
	\item $(h\circ f\circ g)(x)$.
\end{enumerate}
